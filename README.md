# santimendoza_hub

Source for [santimendoza.com](https://santimendoza.com) — Santiago Mendoza's personal site. Built with React, TypeScript, and Vite.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build   # outputs to dist/
```

## Content

Profile, skills, social links, and project data live in `src/data/profile.ts` — edit there, not in the components.
